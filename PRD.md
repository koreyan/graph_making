# Product Requirements Document (PRD): 리플로우 프로파일 데이터 분석 및 어노테이션 도구

## 1. 프로젝트 개요 (Project Overview)
본 프로젝트는 반도체 리플로우(Reflow) 공정 엔지니어가 온도 데이터를 정밀하게 분석하고 보고서용 주석을 달 수 있는 고성능 데스크톱 애플리케이션을 구축하는 데 목적이 있습니다. 

단순한 데이터 표시를 넘어, 특정 공정 구간(Preheat, Soak, Reflow)의 물리적 수치(기울기, 유지 시간 등)를 자동으로 계산하여 엔지니어의 의사결정을 돕는 '인사이트 도구'를 지향합니다.

---

## 2. 기술 스택 (Tech Stack)
현대적인 UI와 정밀한 데이터 시각화를 위해 다음과 같은 스택을 사용합니다.

| 분류 | 기술 | 선정 이유 |
| :--- | :--- | :--- |
| **Framework** | **Electron** | Windows/macOS 크로스 플랫폼 지원 및 풍부한 웹 라이브러리 활용 가능 |
| **Frontend** | **React + TypeScript** | 안정적인 상태 관리 및 타입 안정성 확보를 통한 유지보수성 향상 |
| **Visual** | **Highcharts.js** | 엔지니어링 수준의 정밀한 차트, 줌(Zoom), 마우스 인터랙션 및 어노테이션 기능 내장 |
| **Data** | **PapaParse** | 대용량 CSV 파일의 빠른 브라우저 측 파싱 지원 |
| **Styling** | **Vanilla CSS (Module)** | 디자인 자유도를 최대화하여 `reference.png`의 프리미엄 디자인 구현 |

## 3. 핵심 기능 요구사항 (Functional Requirements)

### 3.1 스마트 데이터 파싱 (Priority: High)
- **비정형 헤더 처리**: CSV 파일 상단의 기기 정보 영역을 인식하여 자동으로 건너뛰고 실제 데이터 시작점부터 파싱.
- **다채널 로딩**: 최대 4개 이상의 온도 채널을 파싱하되, **'열전지 2'**를 기본 분석 대상으로 설정.
- **단위 자동 변환**: 시간 데이터를 경과 시간(0초부터 시작)으로 자동 변환하여 표시.

### 3.2 고성능 인터랙티브 그래프 (Priority: High)
- **멀티 어노테이션 (Reference 반영)**:
    - 마우스 드래그를 통해 특정 영역 선택 시 사각형 박스와 텍스트 라벨 생성.
    - `reference.png`와 같이 특정 온도 구간(예: 217°C, 250°C) 가이드라인 표시.
- **자동 수치 계산**: 선택 영역의 시간 간격(Δt), 온도 변화(ΔT), 그리고 평균 기울기(Slope) 자동 계산.
- **축 및 그리드 설정 (v1.1)**:
    - **배경 그리드**: X, Y축 모두 그리드 라인 표시 (가시성 확보).
    - **X축 (시간)**: 최대 480초, 간격(Tick Interval) 30초 고정.
    - **Y축 (온도)**: 최대 300도, 간격(Tick Interval) 50도 고정.

### 3.3 분석 결과 보존 및 내보내기 (Priority: Medium)
- **보고서용 캡처**: 주석과 통계 수치가 포함된 차트 영역을 이미지로 저장.
- **분석 설정 저장**: 주석 정보를 JSON 파일 등으로 저장/복원.

## 4. 데이터 규격 정의 (Data Specification)
대상 데이터(`T15168 MultiChannel.csv`) 구조:
- **Row 1~7**: 메타데이터 (기기명, 일련번호 등)
- **Row 8~**: 시계열 데이터 (날짜, 시간, 온도 1~4)
- **메인 데이터 선정**: **'열전지 2'** 채널을 분석의 핵심 기준으로 삼아 그래프를 기본 렌더링함.
- **특이사항**: 온도 값이 '-270'인 경우 센서 미연결로 간주하여 제외 처리.

---

## 5. UI/UX 전략
- **Dark & Professional**: 고대비 다크 모드를 기본으로 하여 데이터 가독성 극대화.
- **Drop & Load**: 파일을 드래그 앤 드롭하면 즉시 시각화.
